A collection of "Hello World" programs in various programming languages.

## C

```c
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}
```

## C++

```cpp
#include <iostream>

int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}
```

## Python

```python
print("Hello, World!")
```

## JavaScript

```javascript
console.log("Hello, World!");
```

## Ruby

```ruby
puts "Hello, World!"
```

## Rust

```rust
fn main() {
    println!("Hello, World!");
}
```

## Go

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}
```

## Java

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

## Bash

```bash
echo "Hello, World!"
```

## HTML

```html
<!DOCTYPE html>
<html>
<head>
    <title>Hello World</title>
</head>
<body>
    <h1>Hello, World!</h1>
</body>
</html>
```
